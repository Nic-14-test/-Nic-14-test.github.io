Hi, This is a test on my Raspberry Pi 4

This web page will show you how to set up a web server on your Raspberry Pi 3/4/5.

Step 1: Update your packages

Update Your System: After booting up your Raspberry Pi, open a terminal and update your package lists and upgrade your packages with the following commands:

sudo apt-get update

And

sudo apt-get upgrade

This will make sure that you get the latest versions of every file you download after this.

Step 2: Install Apache

Install Apache, a web server software, by typing the following command in the terminal:

sudo apt-get install apache2 -y

This will allow your Raspberry Pi to answer HTTP requests and serve web pages.

Step 3: Install PHP

Install PHP, a scripting language that can generate dynamic web pages. To do this, type the following command in the terminal:

sudo apt-get install php libapache2-mod-php -y

This will install PHP and configure Apache to use it.

Step 4: Install MariaDB

Install MariaDB, a database management system that can store and retrieve data for your web applications. To do this, type the following command in the terminal:

sudo apt-get install mariadb-server

This will install MariaDB and set up a root user with no password. You can change the password later by running the following command:

sudo mysql_secure_installation

This will also remove some insecure default settings and improve the security of your database.

Step 5: Test your web server

Test your web server by creating a simple HTML file that displays some information. To do this type the following command in the terminal:

cd /var/www/html

To create and open the Home.html file

sudo nano Home.html

Add this code to the Home.html file or make your own code:

            <!DOCTYPE html>
            <html>
            <head>
                <title>Home Page</title>
                <style>
                    #my-container {
                        display: flex;
                        justify-content: center; /* Horizontally center */
                        align-items: center; /* Vertically center */
                        height: 100vh; /* Set the height to the viewport height */
                        background-color: white; /* Set the background color to white */
                    }
                </style>
            </head>
            <body>
                <div id="my-container">
                    <h1>Welcome to my home page!</h1>
                    
                    
                </div>
             </body>
             </html>

Step 6: Restart Apache

To make sure all the changes take effect, restart Apache with the following command:

sudo service apache2 restart

Step 7: View website

Open a web browser on your Raspberry Pi or another device on the same network and type the following address:

http://raspberrypi.local/Home.html

You should see a web page that says "Welcome to my home page! If you see this, congratulations! You have successfully set up a web server on your Raspberry Pi.
